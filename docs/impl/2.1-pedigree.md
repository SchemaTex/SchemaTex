## Scope

Pedigree chart 与 genogram 共享 ~70% 代码。主要区别：

### Parser — fork genogram parser
- 移除 relationship quality (divorce/separation)
- 新增 genetic status: `affected`, `carrier`, `obligate-carrier`, `proband`
- 新增 consanguinity operator `==`

### Layout — simplify genogram layout
- 复用 generation-based layered layout
- 新增：左侧 Roman numeral generation labels (I, II, III)
- 新增：individual numbering within generation (1, 2, 3)
- 移除：divorce/separation line markers

### Renderer — extend genogram renderer
- 新增：proband arrow marker
- 新增：generation labels on left margin
- Modified fill semantics: affected=full, carrier=half, obligate-carrier=dot

---

## Implementation Strategy

尽量复用 genogram 代码。考虑是否值得提取 shared layout utilities 到 `src/core/` 或 `src/diagrams/shared/`。

如果 genogram 和 pedigree 共享代码 > 50%，应该在此步骤做一次 refactor 提取公共部分。
