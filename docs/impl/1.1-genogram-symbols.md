## Goal

实现所有 genogram 标准符号为 SVG 生成函数。每个符号函数接收位置和条件，返回 SVG 字符串。

### SVG Output Structure（per individual）

```xml
<g class="schematex-node schematex-male" data-individual-id="john" data-generation="0"
   transform="translate(100, 50)">
  <title>John (1950)</title>
  <!-- Base shape -->
  <rect x="-20" y="-20" width="40" height="40" class="schematex-shape" fill="white" stroke="black" stroke-width="2"/>
  <!-- Condition fills (if any) -->
  <rect x="-20" y="-20" width="40" height="40" fill="#333" clip-path="url(#half-left-rect)"/>
  <!-- Deceased overlay (if applicable) -->
  <line x1="-20" y1="-20" x2="20" y2="20" stroke="black" stroke-width="2"/>
  <line x1="20" y1="-20" x2="-20" y2="20" stroke="black" stroke-width="2"/>
</g>
```

使用 `src/core/svg.ts` builder，不要拼接 raw string。
