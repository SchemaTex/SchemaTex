## Scope

实现 ecomap diagram type 的全部四件：parser + layout + renderer + integration。

### Parser (`src/diagrams/ecomap/parser.ts`)
- DSL 语法比 genogram 简单：center + systems + connections
- 复用 property parsing 逻辑（可提取到 shared utility）

### Layout (`src/diagrams/ecomap/layout.ts`)
- **Radial/polar layout** — 完全不同于 genogram 的 generation-based
- Center node at (0, 0)
- Systems placed on concentric circles by connection weight
- Minimize overlap between system labels

### Renderer (`src/diagrams/ecomap/renderer.ts`)
- 复用 `src/core/svg.ts` builder
- 新增：wavy line (stressful) SVG path
- 新增：arrowhead markers for energy flow

### Integration
- Register as DiagramPlugin
- Auto-detect via "ecomap" keyword
