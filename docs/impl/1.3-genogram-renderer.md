## Implementation

### Main Function

```ts
export function renderGenogram(layout: LayoutResult, config: RenderConfig): string {
  const defs = buildDefs(layout);
  const styles = buildStyles(config);
  const generations = renderGenerations(layout, config);
  const labels = renderLabels(layout, config);

  return svgRoot({
    viewBox: `0 0 ${layout.width} ${layout.height}`,
    class: 'schematex-diagram schematex-genogram',
    width: layout.width,
    height: layout.height,
  }, [
    title(config.title || 'Genogram'),
    desc(`Genogram with ${layout.nodes.length} individuals`),
    defs,
    styles,
    ...generations,
    labels,
  ]);
}
```

### Rendering Order

1. `<defs>` — fill patterns, clip paths, markers
2. `<style>` — 默认样式（可被外部 CSS 覆盖）
3. Edges — 先画线，再画节点（节点覆盖在线上方）
4. Nodes — individual symbols
5. Labels — 名字 + 年份，最后画（最上层）

### Label Positioning

- 名字显示在 symbol 下方，居中
- 格式：`"Name (birthYear)"` 或 `"Name (birthYear-deathYear)"` if deceased
- 如果 individual 没有 label property，用 ID 作为显示名

## CSS Class Reference

| Class | 用途 |
|-------|------|
| `.schematex-diagram` | SVG root |
| `.schematex-genogram` | 图表类型 |
| `.schematex-generation-N` | 代层 group |
| `.schematex-node` | 任何 individual |
| `.schematex-male` / `.schematex-female` / `.schematex-unknown` | 性别 |
| `.schematex-deceased` | 已故 |
| `.schematex-shape` | 符号 shape 本身 |
| `.schematex-edge` | 关系线 |
| `.schematex-edge-married` / `.schematex-edge-divorced` / ... | 关系类型 |
| `.schematex-label` | 文字标签 |
| `.schematex-sibship-line` | 连接兄弟姐妹的水平线 |
| `.schematex-drop-line` | couple 到 sibship 的垂直线 |
