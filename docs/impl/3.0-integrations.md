## Planned Integrations

### 3.1 Browser renderToElement()

```ts
export function renderToElement(text: string, config?: LineageConfig): SVGElement {
  const svgString = render(text, config);
  const parser = new DOMParser();
  const doc = parser.parseFromString(svgString, 'image/svg+xml');
  return doc.documentElement as unknown as SVGElement;
}
```

需要单独的 browser entry point 或条件导入（避免 Node.js 环境报错 DOMParser）。

### 3.2 React Component

```tsx
// lineage-react (separate package or subpath export)
import { render } from 'lineage';

export function Lineage({ text, config, className }: LineageProps) {
  const svgString = useMemo(() => render(text, config), [text, config]);
  return <div className={className} dangerouslySetInnerHTML={{ __html: svgString }} />;
}
```
